O que são Clients e Servers??

Um client¹ é um dispositivo ou software que solicita serviços ou recursos de um servidor, enquanto um server¹ é um computador ou programa que fornece esses serviços ou recursos.

Como é uma estrutura básica de um HTML?

A estrutura básica de um documento HTML consiste em elementos que organizam o conteúdo, como cabeçalhos, parágrafos, imagens, e outros, e, são delimitados por tags.

Uma nova disposição de elementos para a estruturação em HTML surgiu com a vinda do HTML 5, ou seja, onde antes se utilizava uma sequência de divs para a criação da página web, agora se utiliza as tags header, nav, aside, article, section e footer, por exemplo.

Esse novo modelo de criação traz vantagens no quesito de leitura do site pelos mecanismos de busca da internet, tornando os sites que assim são constuídos, mais suscetíveis a serem apresentados ao usuário da internet.
