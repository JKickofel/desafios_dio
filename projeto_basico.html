Para o que serve a tag "blockquote"?

O blockquote é utilizado para adicionar citações ao texto. Ele deixa o texto com o recuo de uma citação. Veja o exemplo:

"Assim ficará a sua citação."

Nesse exemplo, também foi utilizada a tag "i" para formartar o texto em itálico.

O que são Clients e Servers?

Um client1 é um dispositivo ou software que solicita serviços ou recursos de um servidor, enquanto um server1 é um computador ou programa que fornece esses serviços ou recursos.

Como é uma estrutura básica de um HTML?

A estrutura básica de um documento HTML é a de uma linguagem de marcação. O HTML consiste em elementos que organizam o conteúdo, como cabeçalhos, parágrafos, imagens, e outros, sendo delimitados por tags.

HTML 5: o que mudou?

Uma nova disposição de elementos para a estruturação em HTML surgiu com a vinda do HTML 5, ou seja, onde antes se utilizava uma sequência de divs para a criação da página web, agora se utiliza as tags header, nav, aside, article, section e footer, por exemplo.

A tag "font" por exemplo, foi usada no HTML 4 para especificar o tipo da fonte, seu tamanho e cor do texto. Essa formatação agora é feita em CSS. Veja o exemplo:

Qual a sua vantagem em relação ao HTML anterior?

Esse novo modelo de criação traz vantagens no quesito de leitura do site pelos mecanismos de busca da internet, tornando os sites que assim são constuídos, mais suscetíveis a serem apresentados ao usuário da internet.

Qual mecanismo realiza esta busca pela internet?

O web scraping é o processo de extração de dados de websites de forma automatizada. Ele envolve o uso de robôs ou scripts para simular a navegação humana, coletando informações específicas e estruturando-as para análises posteriores.

Para o que o web scraping é utilizado?

O web scraping é amplamente utilizado em diversas áreas, como comparação de preços, monitoramento de notícias, pesquisa de mercado e coleta de dados governamentais, entre outros.

Entendendo a diferença entre listas ordenadas e não ordenadas

Listas ordenadas e não ordenadas são representadas por ol e ul, respectivamente.

Veja o exemplo de uma lista ordenada (ol):

  1. Item 1
  2. Item 2

Veja o exemplo de uma lista não ordenada (ul):

  • Item 3
  • Item 4

É possível notar que na lista ordenada os ítens são classificados por ordem numérica crescente no exemplo, enquanto na lista não ordenada os ítens são marcados por um símbolo (um bolinha) no exemplo.

Para formatar corretamente essas listas, é necessário fazer uso do CSS.
